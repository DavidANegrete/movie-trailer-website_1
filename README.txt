==========================================
Fresh Tomato Movie player
==========================================

This Python Program displays images of movies along with details about the movie. It includes the Rating, story line, movie image and URL to a youtube preview of the movie.

==========
Files
==========
There are three files included:
    fresh_tomatos.py
    entertainment_center.py
    media.py
